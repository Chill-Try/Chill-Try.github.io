手机很方便，过于方便了

它使你分心，将你从生活中抽离

本文将给出一些技巧，帮你专注当下，回归生活

直击要点：
#

  1. 颜色调黯淡
  2. 开震动模式
  3. 安装 Before Launcher
  4. 外出放在后裤兜里
  5. 卸载花里胡哨的应用
  6. “拥抱” 电脑

接下来，对以上进行说明

1. 颜色调黯淡
#

这里指饱和度，人眼相对偏好色彩鲜艳的画面

例如，本人手机在显示设置中，有 生动 (P3 色域)、标准 (sRGB 色域)、节能 这三档

默认为 生动，在显示设置中，手动将其改为 节能

再激进点，开护眼模式并设置黑白滤镜，不建议，总有人会发来图片的


低饱和颜色使人平静，缓解冲动

用一段时间，手机看起来变得无趣了

对比之下，生活中的色彩会更加生动

再看电脑显示器的 sRGB 色域也顺眼多了


2. 开震动模式
#

目的是减弱提醒。本人手机震动弱，与静音无异

有要事则面谈或电话

发消息的暂搁置，待闲暇时间集中处理

对 “大事小事屁事统统电话者”，降低其优先级（免打扰）


3. 安装 Before Launcher
#

安卓用户可安装此启动器，用它替换你的默认桌面

此桌面十分精简，仅三页：

  1. 中间主页，仅自选放置几个常用 APP
  2. 右页，所有 APP 的列表
  3. 左页，拦截并暂存应用消息，指定白名单放行

中间主页
#

个人而言，作者仅放置了：微信、钉钉、翻译、浏览器、相机、电话

这些 “不会留我很久” 的

值得称赞的是，主页 APP 仅显示名字，不显示彩色图标，让人丧失点进去的欲望

右页
#

所有 APP 的列表，每个 APP 独占一行

每次想打开非 常用 APP，都需手动翻找或打字搜索

繁琐的过程，一定程度上使你远离那些花里胡哨的 APP

同样值得称赞的是，APP 图标是黑白的，让人丧失点进去的欲望

左页
#

保障做事专注与高效，告别被打扰

默认所有 APP 将被过滤，可排除指定 APP

被过滤的 APP 通知不发出提醒，而在此页积攒

有空时集中查看，就像处理邮件一样

消息等人看，而不是人时刻 polling 消息


4. 外出放在后裤兜里
#

外出时，将手机放在后裤兜

放在前兜，手总是碰到它，就想拿出来看看

放在后兜，犹如将烂事抛之脑后

被人偷了怎么办：本人难以想象，是否存在一哺乳动物，被摸臀部时无法察觉，故此题不解


5. 卸载花里胡哨的应用
#

绝大多数手机上能做的，电脑上也能做，Even Better


6. 拥抱电脑
#

用手机做事，随时随地

随时随地，听起来好方便啊！但人类进化了这么久，不随时随地大小便是有原因的

与他人出游时，或许你随时随地掏手机的动作，在对方眼中与随时随地大小便有“异曲同工之妙”


用电脑做事，限制了时间、空间

特定的时间，在特定的地点，做对应的事情

这样习惯后，陪伴他人时，便不再是随时随地掏手机的孽畜
