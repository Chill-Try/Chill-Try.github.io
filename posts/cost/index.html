今天想到的，有关付出与价值

万物有其价值

主观价值随时间、观察对象而变化


情景
#

以下情景是否似曾相识：


获取知识时
#

  • 在短视频看到的，一些认为很棒的理论、知识。只会在短期内影响你
    随即：收藏，忘掉，在收藏夹吃灰

购买商品时
#

负面情况
#

情景 当遭遇负面情况 对等关系
Steam 免费送的游戏： 随便玩儿两下，不太行，随手扔在仓库吃灰 付出 << 价值
花了两三百买的游戏： 这游戏看着不错，但内容不太行，再探索一下吧 付出 ~= 价值
攒了两三百买的游戏： 盼了这么久，以为多好，不太行，退款退款 付出 >> 价值
  • 付出 与 主观价值 在不同的对等关系下，有以上表现

珍惜程度
#

情景 珍惜程度 对等关系
平台免费送的菜 + 十几块的肉： 吃不下倒了，明天换花样 付出 << 价值
几十块的菜 + 几十块的肉： 还剩点儿，撑我事小，得把它消灭掉 付出 ~= 价值
几百的海鲜： 还不错，但吃完只是果腹，下次买便宜的替代品 付出 >> 价值
  • 付出 与 主观价值 在不同的对等关系下，有以上表现

（各人对各物的 主观价值 有不同标准，上述两例仅供参考）

#

总之，想阐述的观点是：

  • 对于获得，付出代价 与 主观价值 越相匹配，越会珍惜

运用
#

它的意义所在？可以用在何处？


知识获取
#

当获取感兴趣的知识时 对等关系
短视频的浓缩内容，看起来很爽很顺畅，但不易吸收 付出 << 价值
优质书籍、长视频使其更易融合于你，成为你的一部分 付出 ~= 价值
外文译本，译者差劲，读起来很辛苦，放弃阅读 付出 >> 价值

一本你喜欢，且通篇文体顺畅的书籍

能使你愉快地完成阅读，并内化成为你的一部分


具有良好逻辑性的视频，有趣，带领你思考

由起始推理出结果，展示其思路，更容易使你接纳

思考，这通常是短视频内容中，占比少甚至不具备的


日常工作
#

上面列的表有点多了，文字描述一下，还是三种关系：

若 水平较差，工作太摆：领导会认为自己吃亏了
若 水平合适，态度良好：理论上最佳
若 态度过于积极：工资不变如此努力，想努力当然有的是工作量

最后一种，虽然我们都不想看到

似乎反人性，但实际上是合乎人性的

何以为然？


不记得从哪里摘录的了，原文是

Some people rush into things without knowing how to do them, only to find themselves in a state of anxiety.

与道德经中的 “为学日益，为道日损。损之又损，以至于无为。无为而无不为。”
在思想上有点相像

这句话挺值得玩味的

（现在想想，小初高就开始学“老子”。。真的算学吗？只能算硬背吧 😐）


  1. 付出 ~= 价值

国外的 WLB (Work-Life Balance)

使工作者能得到良好的休息*，拥有更佳的创造力

也能够抽出时间提升自己的 客观价值

（*休息：尽量放空大脑的前提下
进行 睡眠、冥想、亲近自然、肢体活动，是我对休息的定义）


个人价值提升，能为公司创造更多价值

等你的 客观价值 提升了，能做出更多贡献了

此时可以尝试说服领导，提升他对你的 主观价值
(!! EQ Required)


  1. 付出 >> 价值

玩儿命干提升慢，实属下策下下策

工具毕竟是工具，像人工智能一样的工具，易被其取代

个人认为想象力，亦即创造力最为重要


但为什么，偶尔会遇到，宁愿让你加班当工具

也不愿你提升价值，做好做强的呢


也许，其领导力相对较弱，担心你能力强了，喧宾夺主

此时，需要你在提升 客观价值 的同时

能够认清自己位置，保持谦和（适度的谦逊）


也许，观察对象认为该职位 的 主观价值 低

此时，若你判断现有观察对象，对于该职位的 主观价值 在短期内不会改变

则应该考虑更换观察对象了

（在开头有提到，主观价值 随时间、观察对象而变化）


例如，一段时间内逐渐降低付出，使对等关系趋于 付出 << 价值

向观察对象放出信号的同时，在这段时间内提升自我价值

（有点情商，不要明说得罪人啊，所谓 00 后们）

为寻找 认为该职位主观价值更高的观察对象 做准备


  1. 付出 << 价值

其实已在 付出 >> 价值 中提到了

这也是生活中，有关平衡的映射

极端的人，总是在两极之间摇摆


人际关系
#

日常生活中，我们帮助了某人

某人要表示谢意，我们会客气地回拒对方

客气礼貌，是从小被教育的

目的是人际交往中，使对方感到舒服


婉拒也是拒绝，“婉”只是将程度降低

拒绝对方谢意，谢意，好东西，好心

被拒绝谈不上舒服，好心被拒绝，更谈不上

似乎有违初衷


想要利好他人，也是一种欲望*

在帮助他人，满足自己的“利他欲”后

不妨也让他人满足一下

（*欲望：想得到某种东西或想达到某种目的的要求：求知的～。引自新华词典）


有道理，和本文什么关系？

不舒服是一方面，另一方面，对方觉得欠下了人情

而亏欠使人内疚，只是，较小的人情，感觉不强烈


刚得到对方帮助时，想要回馈

但被拒绝，随着时间推移，主观价值 发生变化

若 变低，不回馈了，曾经的事就算了吧

若 变高，想到就会不安，总觉亏欠，远离，关系恶化


我本人也时常婉拒谢意，总觉给对方添麻烦

应当改变，共勉
